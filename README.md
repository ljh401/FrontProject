# FrontProject
 CSS/HTML , JS COPY 프로젝트 입니다.
 
# 1. 주제
홈페이지 COPY(중미산천문대)

# 2.목적
기존에 존재하는 홈페이지를 COPY하며 Front 기능과 친해지기 위해 해당 프로젝트를 진행하였다.

# 3. 주제선정이유 
중미산 천문대의 경우 JS나 CSS도 처음 front적으로 너무 어렵지 않았기에 css/js에 더 흥미를 가질 수 있을 거라 생각하여 주제를 선정하게 되었다.
